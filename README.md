### supercharged markdown rendering engine

List of features
- [x] Variables

```html
## name: <star>
```


- [x] Loops

```html
<* planet in planets>
```

- [ ] Attributes
- [x] Conditionals

```html
<if star = 'Sun':
    Our solar system

    Number of planets: 8
>
<else if star = 'Kepler-32':
    Cygnus solar system

    Number of planets: 3
>
<else: Some random solar system>
<endif>
```
